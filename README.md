# Mini-Shoppingmall

기술
-----------
HTML,CSS,JS
- json data 사용
- fetch 로 json호출로 서버에 요청, 데이터 받아옴
- 버튼이벤트로 옷 종류, 컬러에 따라 아이템이 필터링됨
- html data-key, value 사용해봄

내용
-------------
좋아하는 유튜버 드림코딩 엘리님의 강의.
javaScript에 대해 계속 공부하던 중 귀여운 예제로 연습할 수 있었다
자바스크립트 개념이 잡혀가는 것 같다 좀 더 연습하다보면 아하! 하는 순간이 올 것 같으니 
더 집중해서 공부해야겠다.
